# Go 语言练习题集

## 难度级别说明
- 🟢 初级：基础语法和概念
- 🟡 中级：数据结构、函数、方法
- 🟠 中高级：接口、错误处理、文件操作
- 🔴 高级：并发编程、反射、设计模式

---

## 🟢 初级练习题 (1-10)

### 1. Hello World 变体

- 输出 "Hello, Go!"

```go
// 预期输出：Hello, Go! 当前时间是：2024-01-01 12:00:00
```
ok

ok

ok

## 🟡 中级练习题 (11-20)

### 11. 结构体和方法
创建一个 `Person` 结构体：
- 包含姓名、年龄、邮箱字段
- 实现方法：自我介绍、过生日、验证邮箱格式
- 创建结构体切片并排序

### 12. 银行账户系统
设计一个银行账户结构体：
- 包含账户号、余额、持有人姓名
- 实现存款、取款、查询余额方法
- 取款时检查余额是否足够
- 记录交易历史

### 13. 学生成绩管理
创建学生成绩管理系统：
- 学生结构体：姓名、学号、成绩map
- 实现添加成绩、计算平均分、查找最高分
- 班级结构体管理多个学生
- 统计班级平均分

### 14. 文件操作练习
编写程序实现：
- 读取文本文件内容
- 写入数据到文件
- 统计文件中的行数、字数、字符数
- 复制文件内容到另一个文件

### 15. JSON处理
编写程序实现：
- 将结构体转换为JSON
- 从JSON字符串解析到结构体
- 处理嵌套JSON结构
- 读取JSON文件并解析

### 16. 错误处理实践
创建一个包含错误处理的程序：
- 自定义错误类型
- 使用errors.New()创建错误
- 实现错误包装和解包
- 设计一个可能失败的函数链

### 17. 接口入门
定义和实现接口：
- 创建Shape接口（面积和周长方法）
- 实现Rectangle和Circle类型
- 使用接口类型作为函数参数
- 接口的多态性演示

### 18. 包和模块
创建自定义包：
- 设计一个工具包（字符串处理、数学运算）
- 导出和非导出函数
- 包的文档注释
- 在主程序中使用自定义包

### 19. 时间和日期处理
编写程序实现：
- 获取当前时间
- 时间格式化和解析
- 计算两个时间的差值
- 定时器和计时器的使用

### 20. 正则表达式
使用正则表达式：
- 验证邮箱格式
- 提取文本中的电话号码
- 替换文本中的特定模式
- 分割字符串

## 📚 学习建议

### 练习顺序
1. 按难度递进：从🟢到🔴
2. 每完成一个难度级别，回顾总结
3. 重点练习自己薄弱的领域
4. 尝试优化已完成的代码

### 编码规范
- 遵循Go官方编码规范
- 使用有意义的变量名
- 编写注释和文档
- 进行错误处理

### 测试和调试
- 为每个练习编写测试
- 使用go test运行测试
- 学会使用调试工具
- 进行性能分析

### 进阶学习
- 阅读Go标准库源码
- 学习开源项目
- 参与社区讨论
- 实践项目开发

### 资源推荐
- Go官方文档：https://golang.org/doc/
- Go语言圣经：https://github.com/golang-china/gopl-zh
- Go实战：https://github.com/goinaction/code
- Effective Go：https://golang.org/doc/effective_go.html
